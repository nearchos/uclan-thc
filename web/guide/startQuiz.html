API Guide: startQuiz

Usage

Initiates a new quiz session for the specified category. The session is automatically created if no session is already available for the given parameters combination.

URL format:
http://uclan-thc.appspot.com/api/csv/startQuiz?playerName=...&appID=...&categoryUUID=...

Parameters

  • playerName: so it can appear on the leader board
  • appID: typically, the name of your team
  • categoryUUID: the UUID of a valid, active category (you can get this by using '/api/csv/categories', see guide:categories).

Possible outcomes

  • When: Everything OK
    Status: "OK", Message: ""
    Second line: the 'session' ID to be used in subsequent calls (e.g. in {@link GetCurrentQuestion} or GetUpdateLocation)
    Third line: the number of questions in the selected category
    Example: 
        OK,
    agtzfnVjbGFuLXRoY3IUCxIHU2Vzc2lvbhiAgICAgMOVCgw
    3
  • When: One or more parameters are missing
    Status: "Invalid or missing parameters", Message: "Valid parameter list 'startQuiz?playerName=...&appID=...&categoryUUID=...'"
    Example: 
        Invalid or missing parameters,Valid parameter list 'startQuiz?playerName=...\
                          &appID=...&categoryUUID=...'
  • When: The category is unknown
    Status: "Unknown category ID", Message: "The specified category ID could not be found"
    Example: 
        Unknown category ID,The specified category ID could not be found
  • When: The category is not active
    Status: "Inactive category", Message: "The specified category is not active currently"
    Example: 
        Inactive category,The specified category is not active right now
  • When: The playerName is already in use
    Status: "Invalid playerName", Message: "The specified playerName is already in use (try a different one)"
    Example: 
        Invalid playerName,The specified playerName is already in use (try a different one)

Issuing this URL

                    &categoryUUID=agtzfnVjbGFuLXRoY3IVCxIIQ2F0ZWdvcnkYgICAgIC6hwoM

returns 
    OK,
